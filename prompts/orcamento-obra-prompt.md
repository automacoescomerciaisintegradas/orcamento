# Prompt para Geração de Orçamento Profissional (SINAPI + BDI)

Utilize este prompt para solicitar a criação de novos orçamentos seguindo a metodologia profissional do sistema.

---
### 🤖 Prompt

"Por favor, elabore um orçamento profissional para o cliente [NOME_CLIENTE]. 
Dados complementares: [CPF/CNPJ], [TELEFONE].

**Escopo da Obra:**
[DESCREVER OS SERVIÇOS AQUI - EXEMPLO: Pintura de 100m² com tinta látex e reparo de 20 blocos de alvenaria].

**Diretrizes Técnicas:**
1. Utilize a base SINAPI para o estado de [ESTADO - EX: CEARÁ].
2. Aplique um BDI de 25% (ou especifique o valor).
3. Organize por categorias: 'Infraestrutura', 'Alvenaria', 'Acabamentos'.
4. Gere o resultado final em formato PDF Premium (Design RQR)."
---

### 📋 Campos Obrigatórios
- **Cliente**: Identificação completa.
- **Escopo**: Descrição mínima dos serviços para levantamento de quantitativos.
- **BDI**: Percentual de despesas indiretas e lucro.
- **UF**: Estado de referência para os preços SINAPI.
